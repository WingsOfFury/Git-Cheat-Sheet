Git Cheat Sheet

  • git init
    • Initialisiert ein lokales Repository in Vorbereitung auf das Verbinden mit einem CodeCommit Repository.

  • git add
    • Fügt dem schwebenden Commit im lokalen Repository alle neuen, geänderten und gelöschten Dateien hinzu.

  • git commit -m

    • Commit mit Kommentar hinzufügen

  • git status
    • Zeigt an, was dem schwebenden Commit im lokalen Repository hinzugefügt wurde oder was nicht.

  • git push
    • Überträgt finalisierte Commits vom lokalen Repository ins CodeCommit Repository, nachdem die Upstream- Nach verfolgungsdaten festgelegt wurden.
